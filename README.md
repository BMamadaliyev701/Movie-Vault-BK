# Movie Vault

Movie Vault - bu premium darajadagi kino va seriallar olami. Foydalanuvchilar eng so'nggi trenddagi kinolarni ko'rishlari, janrlar bo'yicha qidirishlari va o'zlariga yoqqanlarini saqlab qo'yishlari mumkin.

## Xususiyatlari

- **Kuzatish**: Mashhur kinolar, seriallar va trenddagi kontentlarni ko'rish.
- **Qidiruv**: Multi-qidiruv tizimi (kino, serial, o'yin).
- **Janrlar**: Turli janrlar bo'yicha filtrlash (Multfilm, Fantastika, Qo'rqinchli va b.).
- **Sevimlilar**: Yoqtirgan kinolarni `localStorage` ga saqlash imkoniyati.
- **Batafsil ma'lumot**: Har bir kino haqida qisqacha ma'lumot va trailer ko'rish imkoniyati.

## Texnologiyalar

- HTML5 (Templates & Semantics)
- CSS3 (Vanilla CSS, Glassmorphism, Responsive Design)
- JavaScript (Vanilla JS, Fetch API, LocalStorage)
- TMDB API (Kontentlar uchun)
